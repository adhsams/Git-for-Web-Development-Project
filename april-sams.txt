1. What is Semantic HTML? 
   HTML semantic is meant to give your HTML documents meaning so humans and computers can understand its content. 
   It also makes it easier for screen-readers and SEO's to interpret your doc.
    
2. What is HTML used for? 
   It is the structure behind all digital documents displayed on the WWW. It is needed to create any digital document.
    
3. What is an attribute and where do we put it? 
   An attribute can be placed on any element to provide additional information on that element. 
   They are placed in the opening tag of an element in a name="value" pair.    

4. What is the h1 tag used for? How many times should I use it on a page?
   An h1 tag is used to identify the most improtant information on an HTML doc. It should only be used once.
